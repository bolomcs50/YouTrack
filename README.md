# YouTrack

YouTrack is a web app that ingests .csv activity reports from Yuh and turns them into
clear expense tracking with interactive summaries and charts.

## Deployment
At the present moment, YouTrack is hosted [here](https://you-track-nine.vercel.app/) for anyone to use.

## Run locally
If you instead want to run it locally
1. Install dependencies (once): `yarn install`
2. Start the dev server: `yarn dev`
3. Open the URL printed by Vite (usually `http://localhost:5173`)
